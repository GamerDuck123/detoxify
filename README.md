# Readme

To do anything with this sample
1. Change the project name in settings.gradle.kts
2. Update gradle.properties to match, make sure names are also spaceless
3. Update libs.versions.toml to have the proper versions of everything
4. Update the buildSrc's gradle.properties
5. Use the format me.gamerduck.(modid) as the package
6. Go make sure the main files are either (Name)Plugin or (Name)Mod
7. All mixins should be in me.gamerduck.(modid).mixin.mixins and all helpers in me.gamerduck.(modid).mixin
